# Instruções para Adicionar Imagem de Exemplo

Para resolver o problema da imagem que não aparece, siga estas etapas:

1. Adicione uma imagem de exemplo neste diretório (`project/src/assets/`) com o nome `doctor-placeholder.jpg`

2. Você pode usar qualquer imagem de médica que tenha direitos de uso, ou baixar uma imagem de exemplo de um banco de imagens gratuito como Unsplash ou Pexels.

3. Certifique-se de que a imagem tenha um tamanho razoável (recomendado: 800x600 pixels) e esteja no formato JPG.

4. Após adicionar a imagem, o componente Hero.tsx será atualizado para importar esta imagem diretamente, o que deve resolver o problema de exibição.
