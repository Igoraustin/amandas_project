# ⚠️ AÇÃO NECESSÁRIA: Salvar a Foto da Dra. Amanda

Para que a foto da Dra. Amanda apareça no site, você precisa salvar a imagem que compartilhou no local correto:

## Passos Exatos:

1. **Salve a foto que você compartilhou** com o nome exato:
   ```
   dra-amanda.jpg
   ```

2. **Coloque-a neste diretório específico**:
   ```
   project/public/images/dra-amanda.jpg
   ```

3. **Verifique o site** em http://localhost:5173 - a foto deve aparecer imediatamente na seção Hero.

## Verificação:

- O código já foi atualizado para usar este caminho específico
- O servidor de desenvolvimento deve detectar automaticamente a nova imagem
- Se a imagem não aparecer imediatamente, tente atualizar a página do navegador (F5)

## Observação Importante:

A foto deve ser salva exatamente com o nome `dra-amanda.jpg` no diretório `public/images/`. Qualquer outro nome ou localização não funcionará com o código atual.

---

Após salvar a imagem no local correto, a foto da Dra. Amanda será exibida no site, substituindo a imagem temporária.
